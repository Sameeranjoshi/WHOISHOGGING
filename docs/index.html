University of Utah CHPC

Two small scripts. Much less GPU guesswork.

One script finds free GPUs you can actually request. The other shows who is currently using GPU allocations. Host both on GitHub Pages and give your department one stable install path.

Hosted downloads

Download `free_chpc_gpus.sh` Download `whoishogging.sh`

Configure GitHub Pages to publish the docs/ folder. The final URLs will look like https://YOUR-USERNAME.github.io/YOUR-REPO/free_chpc_gpus.sh.

01

Find free GPUs

Show current free GPU slots across clusters, partitions, and accounts you can actually access.

02

See who is using them

List active GPU jobs with users, accounts, partitions, nodes, GPU counts, and a summary by user.
